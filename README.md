### Requisitos

- Maven v3.9 ou superior
- Java Development Kit 17
- Node v18
- NPM v9
- Angular CLI v19
- Docker + Compose
- VSCode

### Instalação

1. Clone o repositório
   ```sh
   git clone git@github.com:RickelmySantos/Login-Angular-SpringSecurity.git
   ```
2. Instale os pacotes NPM
   ```sh
   npm install
   ```
3. suba o container do docker-compose
   ```js
    docker compose up
   ```
